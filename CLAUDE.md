# CLAUDE.md

## Project Overview

Discover Puerto Rico — a travel guide website built with Astro 5, Tailwind CSS 4, deployed to Cloudflare Pages.

## Commands

```bash
npm run dev       # Start dev server at localhost:4321
npm run build     # Production build to ./dist/
npm run preview   # Preview production build locally
```

## Branding

- Colors: Desert Teal #0D7377 (primary), Warm Coral #E8654A (accent)
- Fonts: Outfit (sans), DM Serif Display (serif)
